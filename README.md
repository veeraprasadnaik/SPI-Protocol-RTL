 # SPI Protocol RTL Design

## 📌 Overview
This project presents the design of the SPI (Serial Peripheral Interface) protocol using Verilog.  
The design is developed and verified in Vivado (simulation, synthesis, RTL analysis, floorplanning, and timing analysis) without hardware implementation.

---

## 🛠️ Tools Used
- Vivado  
- Verilog HDL  

---

## 🔌 SPI Signals
- MOSI – Master to Slave  
- MISO – Slave to Master  
- SCLK – Clock signal  
- SS – Slave Select  

---

## ⚙️ Design Flow
1. RTL Design  
2. RTL Analysis  
3. Simulation  
4. Synthesis  
5. Floorplanning  
6. Timing Analysis  

---

## 📁 Files

- spi_slave.v
